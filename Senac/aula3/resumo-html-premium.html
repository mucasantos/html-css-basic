📜 1. Um Pouco da História do HTML

O HTML (HyperText Markup Language) é a linguagem de marcação ferramenta essencial para estruturar a web.

Criado em 1991 por Tim Berners-Lee, evoluiu por várias versões memoráveis:

  • HTML 2.0 (1995): O primeiro padrão global.
  • HTML 3.2 (1997): Trouxe tabelas e melhor suporte a scripts.
  • HTML 4.01 (1999): Estabilizou a web por mais de uma década.
  • XHTML (2000): Uma abordagem mais rígida baseada em XML.
  • HTML5 (Atual): A versão moderna com suporte nativo a áudio, vídeo e semântica real.

🏗 2. Estrutura Básica de um Documento

Todo arquivo nasce deste "esqueleto" fundamental:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

Destaque Profissional: O <!DOCTYPE html> avisa ao navegador que estamos usando a versão mais recente (HTML5).

🏷 3. Principais Tags de Texto

Organize sua hierarquia de títulos e dê ênfase ao conteúdo importante.

<h1>Título Máximo</h1>
<h2>Subtítulo</h2>
<p>Um parágrafo comum.</p>
<strong>Importante (Negrito)</strong>
<em>Ênfase (Itálico)</em>
💎 Dica de Mestre (Avançado):

Use <mark> para destacar textos como se usasse um marca-texto, ou <abbr> para abreviações como HTML.

📊 5. Tabelas HTML

Utilizadas para exibir dados organizados em linhas e colunas.

Tag Função
<table> Container da tabela
<tr> Linha (Table Row)
<th> Cabeçalho (Table Header)
<td> Célula (Table Data)

📝 6. Formulários e Inputs

O coração da interatividade. Coleta dados e preferências dos usuários.

<form>
  <label>Nome:</label>
  <input type="text">
  
  <label>Email:</label>
  <input type="email">
  
  <button type="submit">Enviar</button>
</form>

Tipos de Input Essenciais:

  • type="text": Texto simples
  • type="password": Senha oculta
  • type="email": Valida e-mail
  • type="radio": Opção única
  • type="checkbox": Múltiplas opções
💎 Dica de Mestre (Tipos Modernos):

Use type="date" para calendários, type="color" para seletores de cor e type="range" para controles deslizantes.

🌿 7. Estrutura Semântica (HTML5)

Fundamental para o Google entender seu site e para a acessibilidade.

<header> Topo do site
<nav> Menu principal
<main> Conteúdo único
<section> Seção temática
<article> Post ou card
<footer> Rodapé

📌 8. Atributos Importantes

  • href: Destino de links (<a>)
  • src: Origem de imagens (<img>)
  • alt: Texto alternativo (Acessibilidade)
  • target="_blank": Abre link em nova aba
  • required: Campo obrigatório em formulários
