Guia Completo: CSS Básico

Layout, Box Model e Posicionamento

📜 1. O que é CSS?

O CSS (Cascading Style Sheets) controla como os elementos HTML aparecem na tela. Ele cuida do design, das cores, das fontes e do layout (espaçamentos e posições).

🏗 2. Sintaxe e Regras

seletor {
  propriedade: valor;
}

Exemplo: h1 { color: red; font-size: 20px; }

📦 3. O Box Model: Padding vs Margin

Todo elemento no CSS é uma "caixa". Entender a diferença entre Padding e Margin é essencial:

CONTEÚDO

📏 Padding (Interno)

O espaço dentro da borda. Ele afasta o conteúdo da sua própria borda.

padding: 20px;

📏 Margin (Externo)

O espaço fora da borda. Ele empurra outros elementos para longe desta caixa.

margin: 20px;

Shorthand (Atalhos)

Você pode definir os 4 lados de uma vez seguindo a ordem: Topo, Direita, Baixo, Esquerda (sentido horário).

/* Topo | Direita | Baixo | Esquerda */
margin: 10px 20px 30px 40px;

/* Topo e Baixo | Direita e Esquerda */
padding: 10px 20px;

📍 4. Propriedade Position

Define como um elemento deve ser posicionado na página.

Valor Descrição
static Padrão. Segue o fluxo normal da página.
relative Posicionado relativo à sua posição original.
absolute Posicionado relativo ao primeiro elemento pai com position.
fixed Fica "preso" em relação à janela do navegador (não rola).
sticky Fica "preso" conforme você rola a página, mas dentro do seu container.
💎 Dica de Mestre:

Ao usar absolute ou relative, você passa a ter acesso às propriedades top, bottom, left e right para mover o elemento.

🎯 5. Exercícios Práticos

1. Desafio do Espaçamento: Crie uma caixa com 200px de largura, borda de 2px sólida preta, fundo amarelo e um padding de 40px. Como o texto dentro dela se comporta?
2. Margens Automáticas: Se você definir margin-left: auto; e margin-right: auto; em uma caixa que tem uma largura (width) fixa, o que acontece com ela na tela?
3. O Topo Fixo: Como você faria para deixar um menu (tag <nav>) sempre visível no topo da tela, mesmo quando o usuário rolar a página para baixo?
4. Classes e IDs: Crie um parágrafo com uma classe chamada "aviso" que tenha a cor do texto vermelha. Depois, crie um ID chamado "destaque" que mude o fundo para azul.
