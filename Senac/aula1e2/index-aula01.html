O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). Ao contrário do que muitos pensam, não é uma linguagem de programação, mas sim uma linguagem de marcação utilizada para estruturar páginas na web.

Com o HTML, organizamos textos, imagens e links. Ele usa tags para marcar cada parte do conteúdo, permitindo que os navegadores entendam o que deve ser exibido como um título, um parágrafo ou uma lista.

Estrutura de uma Página Semântica

A semântica é fundamental para a acessibilidade e para que motores de busca (como o Google) entendam seu site.

Glossário de Termos

Tags
Os comandos básicos do HTML, como <h1> ou <p>.
Atributos
Informações extras passadas dentro de uma tag, como o src de uma imagem.
Semântica
O uso de tags que dão significado ao conteúdo, facilitando a leitura por máquinas.

Comparativo de Tags Comuns

Muitas pessoas confundem tags que parecem iguais visualmente, mas têm significados diferentes:

Diferença entre tags Visuais e Semânticas
Tag Tipo Significado / Uso
<b> Visual Apenas deixa o texto em negrito sem ênfase extra.
<strong> Semântica Indica que o texto é importante (ênfase forte).
<i> Visual Apenas deixa o texto em itálico.
<em> Semântica Indica ênfase enfatizada (frequentemente lida com entonação diferente).

Como começar?

Siga estes passos para criar sua primeira página:

  1. Escolha um editor de código (VS Code, por exemplo).
  2. Crie a estrutura básica:
    • Declare o <!DOCTYPE html>
    • Abra a tag <html>
    • Defina o <head> e o <body>
  3. Adicione conteúdo usando as tags que aprendemos.
  4. Salve com a extensão .html.
